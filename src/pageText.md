# Hello!

For convenience, the text on the pages of this site are completely controlled by this text file (pageText.md). You can edit the text here, and it will automatically update on the website! To edit text for a specific page, simply locate the page title on this file and edit the text beneath it.

The text here is written in a style called Markdown, which makes it easy to style the text on the website. Here are some brief instructions on how to use markdown.

First and foremost, there are three sizes of text.

# The biggest text, called a header

In this case, it's header 1: the biggest style of text. This is styled as purple on the website. You can easily denote header 1 using a hashtag followed by your text: `# Text here`.

## Second biggest, header 2

Likewise, you can make a slightly smaller header (header 2) using two hashtags: `## Text here`.

### Finally, the smallest header

This is header 3. As you may have guessed, you can make it using `### Text here`.

The smallest text is normal text. Conveniently, this is the default size of text, and requires no special symbols! Just start typing, and the text will be formatted automatically.

It's not required, but it's recommended to put one empty line in between each paragraph, and between headers, for readability like so:

```
# header

A paragraph here

Another paragraph, all spaced with empty lines
```

# Advanced techniques

That's the most important things covered, but if you want to get fancy there's a few more things you can do with markdown.

## Links

Perhaps the most convenient advanced topic is a link. For instance, here's a link to [Google](https://google.com). However, be warned, as links written in markdown will open the new site on the same tab, effectively closing your website.

A link has two components: the text and the url. The text is what's presented to the user as blue with an underline (in this case, Google), while the url is the location that you're linking to (in this case https://google.com). As you can see, urls are also automatically formatted as links, but to link plain text you can use `[text here](link here)`. So a link to [Google](https://google.com) would be written as `[Google](https://google.com)`.

You can also link to a page on your own website. All you need to do is use a forward slash and the page you want to go to. A link to the [process page](/process) would look like this: `[process page](/process)`. To link to the [home page](/), use a slash with nothing after it, like this: `[home page](/)`.

## Bold

If you need to emphasis something it's very easy. You can just use **bold text**. You can make text bold by surrounding it with **, for instance `**text here**`.

## Lists

If you want, you can even make lists in markdown!

1. It's pretty much what you'd expect

2. Just use numbers to denote the item

3. It should look like this:

```
1. The first item

2. The second item

3. The third item
```

You can also make bullet points!

- It's very similar to a list

- Just swap the numbers for dashes

- It should look like this:

```
- The first item

- The second item

- The third item
```

To put everything into practice, here's a full example with every style; the formatted text followed by the markdown for it:

# The Example

This is the example. From here we can link to [Google](https://google.com) with the url https://google.com.
